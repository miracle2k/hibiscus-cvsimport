<form>
	<p>
		<span color="header" font="header">Einstellungen</span>
	</p>

	<li>
		Wenn Sie über eine dauerhafte Internetverbindung verfügen,
		können Sie die Option <b>keine Nachfrage vor Verbindungsaufbau</b> aktivieren.
	</li>
	
	<li>
		Sie können ein Limit für den Maximalbetrag einer Überweisung
		definieren um zu verhindern, dass Sie versehentlich einen
		zu grossen Betrag eingeben.
	</li>
	
	<li>
		Bei aktivierter PIN-Prüfung wird aus der von Ihnen eingegebene PIN
		eine Check-Summe gebildet und diese mit der Check-Summe Ihrer ersten PIN-Eingabe
		verglichen. Hierbei wird nicht die PIN selbst gespeichert sondern lediglich die
		Prüfsumme mit der ermittelt werden kann, ob Ihre aktuelle Eingabe mit der Erst-Eingabe
		übereinstimmt.
	</li>
	
	<li>
	  Wenn Sie die Option <b>TANs während der Eingabe anzeigen</b> aktivieren,
	  werden die TAN-Nummern während der Eingabe im Klartext angezeigt, andernfalls
	  wie die PIN gegen Sterne ersetzt.
	</li>
	
	<li>
	  Mit der Option <b>Tausender-Trennzeichen bei Geld-Beträgen verwenden</b>
	  können Sie entscheiden, ob bei Geld-Beträgen ein Tausender-Punkt angezeigt
	  werden soll.
	</li>
	
	<li>
	  Deaktivieren Sie die Option <b>Kontonummern via Prüfsumme der Bank testen</b>,
	  wenn Sie nicht wünschen, dass Hibiscus die von Ihnen eingegebenen Kontonummern
	  auf Plausibilität prüfen soll.
	</li>
</form>
