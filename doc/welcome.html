Hibiscus Dokumenation

Hibiscus ist eine Online-Banking Anwendung für das deutsche HBCI Protokoll. Sie basiert auf der HBCI Implementierung HBCI4Java und läft als Plugin innerhalb des Frameworks Jameica.

Support

Bei Problemen mit der Installation oder Einrichtung von Hibiscus schauen Sie einfach in das Wiki. Dort finden Sie unter anderem:
  • eine FAQ mit Antworten zu den häufigsten Fehlern,
  • eine Liste der getesteten Banken,
  • sowie die bisher getesteten Chipkartenleser
Sollten Sie keine Lösung für den Fehler gefunden haben, dann wenden Sie sich bitte an das Forum www.onlinebanking-forum.de.
Beachten Sie beim Melden von Fehlern bitte grundsätzlich folgende Hinweise:
  • Beschreiben Sie den Fehler möglichst exakt - am besten mit dem wortwörtlichen Fehlertext oder einem Screenshot.
  • Geben Sie die Versionsnummer von Hibiscus und Jameica sowie das von Ihnen verwendete Betriebssystem an.
  • Senden Sie nach Möglichkeit die erzeugte Log-Datei "jameica.log" im Anhang mit. Sie finden die Datei in Ihrem Benutzerverzeichnis:
    Windows C:\Dokumente und Einstellungen\<benutzername>\.jameica
    Linux/Linux-AMD64 /home/<benutzername>/.jameica
    MacOS .jameica (im Benutzerverzeichnis)
    Öffnen Sie die Datei bitte vor dem Senden noch in einem Texteditor und schwärzen Sie ggf. vorhandene persönliche und sensible Daten durch das Zeichen "X" aus.
Falls Sie einen Fehler in Hibiscus gefunden haben oder Sie sich eine Verbesserung wünschen, dann schauen Sie in das Bugtracking-System Bugzilla unter http://www.willuhn.de/bugzilla/. Sollte Ihr Fehler oder Wunsch dort noch nicht erfasst sein, dann können Sie sich selbst einen Bugzilla-Account einrichten und einen neuen Bug-Report verfassen. Beachten Sie hierbei ebenfalls die oben stehenden Hinweise. Sie werden automatisch per Mail informiert, wenn der Report bearbeitet wird.
